# 📊 Exam Score Prediction using Machine Learning

## 📌 Project Overview

This project applies supervised machine learning techniques to predict student exam scores based on academic, behavioral, and environmental factors.

## 📂 Dataset

* Source: Kaggle
* Link: https://www.kaggle.com/datasets/kundanbedmutha/exam-score-prediction-dataset
* Description: Contains student features such as study hours, attendance, sleep patterns, and exam difficulty.

## 🎯 Objective

To build and evaluate regression models that accurately predict student exam performance.

## ❓ Research Questions

1. Baseline model performance
2. Best-performing model
3. Effect of preprocessing
4. Feature importance
5. Sensitivity to evaluation metrics
6. Model robustness
7. Final model selection

## ⚙️ Methodology

* Data preprocessing (encoding, scaling)
* Train-test split (80/20)
* Model training and evaluation
* Feature importance analysis
* Cross-validation and robustness testing

## 🤖 Models Used

* Linear Regression
* Decision Tree
* k-NN
* Random Forest
* Support Vector Regressor (SVR)
* XGBoost

## 📏 Evaluation Metrics

* MAE (Mean Absolute Error)
* RMSE (Root Mean Squared Error)
* R² Score

## 📊 Results

* Tables and figures are available in the `results/` directory
* Each research question has corresponding outputs
